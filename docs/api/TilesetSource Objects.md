ETileset Source

**C++ Source:**

- **Plugin**: CesiumForUnreal
- **Module**: CesiumRuntime
- **File**: Cesium3DTileset.h

<a id="unreal.TilesetSource.FROM_CESIUM_ION"></a>

#### FROM\_CESIUM\_ION

0: The tileset will be loaded from Cesium Ion using the provided IonAssetID
and IonAccessToken.

<a id="unreal.TilesetSource.FROM_URL"></a>

#### FROM\_URL

1: The tileset will be loaded from the specified Url.

<a id="unreal.TilesetSource.FROM_ELLIPSOID"></a>

#### FROM\_ELLIPSOID

2: The tileset will be loaded from the georeference ellipsoid.
