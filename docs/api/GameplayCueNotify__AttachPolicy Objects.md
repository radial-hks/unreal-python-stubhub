EGameplayCueNotify_AttachPolicy

    Specifies if the gameplay cue notify should attach to the target actor.

**C++ Source:**

- **Plugin**: GameplayAbilities
- **Module**: GameplayAbilities
- **File**: GameplayCueNotifyTypes.h

<a id="unreal.GameplayCueNotify_AttachPolicy.DO_NOT_ATTACH"></a>

#### DO\_NOT\_ATTACH

0: Do not attach to the target actor.  The target may still be used to get location and other information.

<a id="unreal.GameplayCueNotify_AttachPolicy.ATTACH_TO_TARGET"></a>

#### ATTACH\_TO\_TARGET

1: Attach to the target actor if possible.
