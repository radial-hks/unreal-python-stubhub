## RoofShape Objects

```python
class RoofShape(EnumBase)
```

建筑屋顶类型

**C++ Source:**

- **Plugin**: AesWorld
- **Module**: AesAsset
- **File**: AesAssetCommon.h

<a id="unreal.RoofShape.FLAT"></a>

#### FLAT

0: 平屋顶

<a id="unreal.RoofShape.GABLED"></a>

#### GABLED

1: 山墙屋顶，也称为尖顶或斜屋顶

<a id="unreal.RoofShape.HIPPED"></a>

#### HIPPED

2: 四坡屋顶

<a id="unreal.RoofShape.PYRAMIDAL"></a>

#### PYRAMIDAL

3: 金字塔状的帐篷屋顶，没有中央的主山脊，只有一个两侧交汇的高点

<a id="unreal.RoofShape.SKILLION"></a>

#### SKILLION

4: 单坡屋顶，它没有中间的主山脊。与平屋顶相比，它的一边比另一边高

<a id="unreal.RoofShape.HALF_HIPPED"></a>

#### HALF\_HIPPED

5: 半坡屋顶

<a id="unreal.RoofShape.HIPPED_AND_GABLED"></a>

#### HIPPED\_AND\_GABLED

6: 歇山顶

<a id="unreal.RoofShape.BUTTERFLY"></a>

#### BUTTERFLY

7: 蝶形屋顶，斜坡在最低点相交的屋顶形状。

<a id="unreal.RoofShape.ROUND"></a>

#### ROUND

8: 圆屋顶，横截面为半个圆，有时也称为桶形屋顶

<a id="unreal.RoofShape.GAMBREL"></a>

#### GAMBREL

9: 斜折屋顶

<a id="unreal.RoofShape.MANSARD"></a>

#### MANSARD

10: 双重斜坡屋顶

<a id="unreal.RoofShape.DOME"></a>

#### DOME

11: 圆顶

<a id="unreal.RoofShape.ONION"></a>

#### ONION

12: 洋葱状屋顶

<a id="unreal.RoofShape.SAWTOOTH"></a>

#### SAWTOOTH

13: 锯齿状屋顶

<a id="unreal.RoofShape.CONE"></a>

#### CONE

14: 圆锥屋顶

<a id="unreal.RoofShape.CROSSPITCHED"></a>

#### CROSSPITCHED

15: 交叉倾斜屋顶

<a id="unreal.RoofShape.SIDE_HIPPED"></a>

#### SIDE\_HIPPED

16: 一半是坡屋顶，一半是山墙

<a id="unreal.RoofShape.SIDE_HALF_HIPPED"></a>

#### SIDE\_HALF\_HIPPED

17: 一半是半坡屋顶，一半是山墙

<a id="unreal.RoofShape.GABLED_HEIGHT_MOVED"></a>

#### GABLED\_HEIGHT\_MOVED

18: 形状是山墙，但在两个部分之间有垂直偏移

<a id="unreal.RoofShape.NONE"></a>

#### NONE

19: 空类型，表示数据无该属性
