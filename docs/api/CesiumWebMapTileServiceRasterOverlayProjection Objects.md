## CesiumWebMapTileServiceRasterOverlayProjection Objects

```python
class CesiumWebMapTileServiceRasterOverlayProjection(EnumBase)
```

Specifies the type of projection used for projecting a Web Map Tile Service
raster overlay.

**C++ Source:**

- **Plugin**: CesiumForUnreal
- **Module**: CesiumRuntime
- **File**: CesiumWebMapTileServiceRasterOverlay.h

<a id="unreal.CesiumWebMapTileServiceRasterOverlayProjection.WEB_MERCATOR"></a>

#### WEB\_MERCATOR

0: The raster overlay is projected using Web Mercator.

<a id="unreal.CesiumWebMapTileServiceRasterOverlayProjection.GEOGRAPHIC"></a>

#### GEOGRAPHIC

1: The raster overlay is projected using a geographic projection.
