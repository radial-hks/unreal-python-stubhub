## PixelStreaming51CloudAudioComponent Objects

```python
class PixelStreaming51CloudAudioComponent(SynthComponent)
```

Allows in-engine playback of incoming WebRTC audio from a particular Pixel Streaming for 51Cloud player/peer using their mic in the browser.
Note: Each audio component associates itself with a particular Pixel Streaming for 51Cloud player/peer (using the the Pixel Streaming for 51Cloud player id).

- **File**: PixelStreaming51CloudAudioComponent.h
