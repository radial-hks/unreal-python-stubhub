## ImGuiCanvasSizeType Objects

```python
class ImGuiCanvasSizeType(EnumBase)
```

EIm Gui Canvas Size Type

**C++ Source:**

- **Plugin**: UnrealImGui
- **Module**: ImGui
- **File**: ImGuiModuleSettings.h

<a id="unreal.ImGuiCanvasSizeType.CUSTOM"></a>

#### CUSTOM

0: Canvas will have a custom width and height.

<a id="unreal.ImGuiCanvasSizeType.DESKTOP"></a>

#### DESKTOP

1: Canvas will have the same width and height as the desktop.

<a id="unreal.ImGuiCanvasSizeType.VIEWPORT"></a>

#### VIEWPORT

2: Canvas will always have the same width and height as the viewport.
