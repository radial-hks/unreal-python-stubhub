## FollowingMode Objects

class FollowingMode(EnumBase)

跟踪模式的状态

**C++ Source:**

- **Plugin**: WdpCamera
- **Module**: WdpCamera
- **File**: WdpDataSource.h

<a id="unreal.FollowingMode.FM_NONE"></a>

#### FM\_NONE

0: 无追踪

<a id="unreal.FollowingMode.FM_TRACKING_LOCKED"></a>

#### FM\_TRACKING\_LOCKED

1: 锁定跟踪，操作不会脱离跟踪对象

<a id="unreal.FollowingMode.FM_TRACKING_LIGHT"></a>

#### FM\_TRACKING\_LIGHT

2: 非锁定跟踪，任何点击操作都可以打断跟踪
