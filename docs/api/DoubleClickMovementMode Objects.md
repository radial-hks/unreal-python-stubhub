class DoubleClickMovementMode(EnumBase)

双击鼠标的移动模式

**C++ Source:**

- **Plugin**: WdpCamera
- **Module**: WdpCamera
- **File**: WdpDataSource.h

<a id="unreal.DoubleClickMovementMode.DCM_DISABLE"></a>

#### DCM\_DISABLE

0: 禁用双击移动

<a id="unreal.DoubleClickMovementMode.DCM_DIRECT_MOVEMENT"></a>

#### DCM\_DIRECT\_MOVEMENT

1: 直接移动到点击位置

<a id="unreal.DoubleClickMovementMode.DCM_PLANE_ONLY"></a>

#### DCM\_PLANE\_ONLY

2: 只能在平面空间移动

<a id="unreal.DoubleClickMovementMode.DCM_SURFACE_ONLY"></a>

#### DCM\_SURFACE\_ONLY

3: 只能点击表面移动

<a id="unreal.DoubleClickMovementMode.DCM_NAV_MESH"></a>

#### DCM\_NAV\_MESH

4: 使用Navmesh移动
